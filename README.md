# vpc_connectivity
Terraform-based AWS VPC deployment featuring multi-AZ public/private subnets, EC2 instances, routing, and Infrastructure as Code automation.
Designed and automated a custom AWS network infrastructure using Terraform, featuring a multi-AZ Virtual Private Cloud (VPC) with public and private subnets, route tables, Internet Gateway connectivity, security groups, and EC2 deployment. This project demonstrates Infrastructure as Code (IaC), AWS networking fundamentals, cloud security principles, and automated environment provisioning using Terraform and GitHub.
